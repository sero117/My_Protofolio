Cafe_Store App

Cafe_Store App is a mobile application built with Firebase Authentication for secure user login and Stripe API integration for safe online payments. The app provides a smooth shopping experience with a modern UI and real-time order tracking.

Tools & Technologies

  • Framework: Flutter
  • Backend: Firebase Authentication & Firestore
  • Payments: Stripe API
  • UI: Custom Flutter Widgets
