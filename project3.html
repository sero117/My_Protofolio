MyAppointments App

MyAppointments App is a scheduling application built with Flutter and Firebase Firestore. It supports push notifications, appointment reminders, and real-time data synchronization. The app provides a seamless experience for users to book and manage appointments efficiently.

Tools & Technologies

  • Framework: Flutter
  • Backend: Firebase Firestore
  • Notifications: Firebase Cloud Messaging
  • UI: Custom Flutter Widgets
