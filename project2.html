Social_Dating UI

This project is a modern dating app UI designed in Figma and implemented using TailwindCSS. It focuses on responsive layouts, smooth animations, and a clean user experience.

Tools & Technologies

  • Design: Figma
  • Frontend: TailwindCSS
  • Animations: CSS Transitions & Hover Effects
  • Responsive Layouts: Flexbox & Grid
